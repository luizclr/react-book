## Quick Start

### Running

```bash
# Install packages
yarn install
# Start server locally
yarn start
```

Server will be opened automatically on [localhost:8080](http://localhost:8080).

### Coding style

```bash
# Linter
yarn run lint
```

### Build

```bash
# Build for production
yarn run buld
```
